# lora_slm — Pipeline LoRA par agent Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Créer `~/Secretarius/lora_slm/`, pipeline automatisé `./run.sh tiron` → train → merge → GGUF pour fine-tuner phi-4-mini-instruct comme routeur Tiron.

**Architecture:** Les scripts `lora_train.py` et `merge_and_quantize.py` sont repris de `https://github.com/mauceri/lora_local_AMD` (validés sur sanroque pour l'extracteur phi-4-mini) avec adaptations minimales. `run.sh AGENT` orchestre les deux étapes en sourçant `agents/<AGENT>/config.env`. Les tests s'exécutent dans le venv ROCm du projet (`lenv/`).

**Tech Stack:** Python 3.9+, PyTorch 2.5.1 ROCm 6.2, HuggingFace transformers + peft + datasets + accelerate, llama.cpp (`~/llama.cpp/`).

## Global Constraints

- Tous les fichiers dans `~/Secretarius/lora_slm/`
- Venv dédié : `lora_slm/lenv/` (créé depuis `requirements-rocm.txt`, exclus du dépôt)
- Tests lancés avec `lora_slm/lenv/bin/python -m pytest tests/ -v`
- `PYTORCH_HIP_ALLOC_CONF=expandable_segments:True` et `HSA_OVERRIDE_GFX_VERSION=10.3.0` posés dans `lora_train.py` avant tout import torch (déjà présents dans lora_local_AMD — ne pas supprimer)
- Modèle de base : `~/Modèles/phi4/` (poids HF phi-4-mini-instruct, local)
- llama.cpp : `~/llama.cpp/` (`build/bin/llama-quantize` et `convert_hf_to_gguf.py` présents)
- Artefacts exclus du dépôt : `checkpoints/`, `merged/`, `*.gguf`, `lenv/`, `*.log`
- Un seul commit par tâche

```python
import sys
from pathlib import Path
sys.path.insert(0, str(Path(__file__).parent.parent))

from datasets import DatasetDict, Dataset


def _ds(messages_list):
    return DatasetDict({"train": Dataset.from_list(
        [{"messages": m} for m in messages_list]
    )})


def test_build_text_dataset_chatML():
    from lora_train import build_text_dataset
    ds = _ds([[
        {"role": "system",    "content": "Routeur Tiron."},
        {"role": "user",      "content": "capture https://example.com"},
        {"role": "assistant", "content": '{"command": "/c", "args": "https://example.com"}'},
    ]])
    result = build_text_dataset(ds, "", "", num_proc=1)
    text = result["train"][0]["text"]
    assert "<|system|>: Routeur Tiron." in text
    assert "<|user|>: capture https://example.com" in text
    assert '<|assistant|>: {"command": "/c"' in text


def test_build_text_dataset_null_command():
    from lora_train import build_text_dataset
    ds = _ds([[
        {"role": "system",    "content": "Routeur Tiron."},
        {"role": "user",      "content": "commande une pizza"},
        {"role": "assistant", "content": '{"command": null, "args": ""}'},
    ]])
    result = build_text_dataset(ds, "", "", num_proc=1)
    text = result["train"][0]["text"]
    assert '"command": null' in text


def test_build_text_dataset_rejects_unknown_format():
    import pytest
    from lora_train import build_text_dataset
    ds = DatasetDict({"train": Dataset.from_list([{"unknown": "x"}])})
    with pytest.raises(ValueError):
        build_text_dataset(ds, "", "", num_proc=1)
```

- [ ] **Step 4 : Créer run.sh**

Créer `~/Secretarius/lora_slm/run.sh` avec ce contenu :

```bash
#!/usr/bin/env bash
set -euo pipefail

AGENT=${1:?Usage: ./run.sh <agent>}
DIR="$(cd "$(dirname "$0")" && pwd)"
source "$DIR/agents/$AGENT/config.env"

mkdir -p "$OUTPUT_DIR" "$MERGED_DIR"

echo "=== [1/2] Entraînement LoRA — $AGENT ==="
"$DIR/lenv/bin/python" "$DIR/lora_train.py" \
  --model_path       "$MODEL_PATH" \
  --data_file        "$CORPUS_TRAIN" \
  --output_dir       "$OUTPUT_DIR" \
  --max_len          "${MAX_LEN}" \
  --epochs           "${EPOCHS}" \
  --lora_r           "${LORA_R}" \
  --lora_alpha       "${LORA_ALPHA}" \
  --lr               "${LR}" \
  --per_device_batch "${PER_DEVICE_BATCH}" \
  --grad_accum       "${GRAD_ACCUM}" \
  --log_file         "$OUTPUT_DIR/training.log"

echo "=== [2/2] Fusion + export GGUF — $AGENT ==="
"$DIR/lenv/bin/python" "$DIR/merge_and_quantize.py" \
  --base           "$MODEL_PATH" \
  --lora           "$OUTPUT_DIR" \
  --out            "$MERGED_DIR" \
  --gguf-dir       "$GGUF_DIR" \
  --llama-cpp      "$LLAMA_CPP_PATH" \
  --quantize-types $QUANTIZE_TYPES

mv "$GGUF_DIR/model-Q6_K.gguf" "$GGUF_DIR/$AGENT-router-Q6_K.gguf"
echo "=== GGUF produit : $GGUF_DIR/$AGENT-router-Q6_K.gguf ==="
```
