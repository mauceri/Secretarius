# shared

Espace mutualise pour les ressources reutilisables entre la generation DSPy/GEPA et le fine-tuning LoRA.

- `prompts/` : prompts communs
- `scripts/` : utilitaires partages
- `schemas/` : schemas et formats de reference
- `datasets/source/` : donnees brutes de reference
- `eval/configs/` : configurations d'evaluation reutilisables
