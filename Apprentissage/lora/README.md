# lora

Chantier dedie au fine-tuning LoRA et a l'evaluation.

- `prompts/` : prompts d'inference et de validation
- `notebooks/` : exploration d'entrainement et analyse
- `scripts/` : preparation, entrainement, evaluation
- `datasets/prepared/` : jeux prets pour le fine-tuning
- `eval/configs/` : protocoles et configurations d'evaluation
- `eval/reports/` : resultats et rapports
- `artefacts/checkpoints/` : checkpoints LoRA
- `artefacts/logs/` : journaux d'execution
- `artefacts/exports/` : exports finaux
